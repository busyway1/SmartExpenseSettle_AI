💡 프롬프트 예시

  • • 자동분류 - 문서 유형별로 자동 분류하여 각각의 정보를 추출해주세요 (⚡ 비동기 병렬 처리로 빠른 분석)
  • • 이 계약서에서 계약 당사자 정보와 계약 기간을 추출해주세요
  • • 영수증에서 구매한 품목들을 카테고리별로 분류해주세요
  • • 견적서의 총 금액이 100만원 이상인 품목만 찾아주세요
  • • 이 문서에서 법적으로 중요한 조항들을 찾아 요약해주세요
  • • 회계 처리를 위해 필요한 모든 정보를 추출해주세요
🎯 자동분류 기능: 여러 문서 유형(수출신고필증, 세금계산서, 인보이스, BL 등)이 섞여있는 경우, "자동분류"를 입력하면 각 문서 유형별로 자동 인식하여 해당하는 정보를 추출합니다. ⚡ 비동기 병렬 처리로 여러 문서를 동시에 분석하여 속도가 향상됩니다.

OpenAI API 키 설정

AI 분석 기능을 사용하려면 OpenAI API 키가 필요합니다.
환경 변수 OPENAI_API_KEY에 API 키를 설정해주세요.

설정 방법:
• Windows: set OPENAI_API_KEY=your-api-key-here
• 또는 코드에서 직접 설정: analyzer = AIAnalyzer(api_key="your-key")
